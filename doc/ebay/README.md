# eBay Integration Docs

Docs in this folder govern how the project interacts with eBay Developer APIs.

## Contents

- `INTEGRATION_POLICY.md` — our Marketplace Account Deletion stance (opted
  out), the exemption reason we submit, the allow-list and block-list of
  fields we may cache, cache TTLs, and the triggers that would force us to
  switch from opt-out to subscribed.

More docs will land here as the eBay integration work progresses (e.g. API
client notes, query patterns, rate-limit strategy, prototype findings).
